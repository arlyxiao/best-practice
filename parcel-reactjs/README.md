# parcel-reactjs

**It's coming with these packages:**
- `Parcel`
- `Reactjs`
- `Babel`
- `Prettier`
- `Typescript`
- `Scss`


## Building and running on localhost

First install dependencies:

```sh
npm install
```

To run in hot module reloading mode:

```sh
npm start
```

To create a production build:

```sh
npm run build-prod
```

## Running

Open http://localhost:1234
